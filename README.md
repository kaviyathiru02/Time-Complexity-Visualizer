# Time Complexity Visualizer Using Flask

## Objective
To develop a web application that visualizes the growth of common algorithmic time complexities such as O(1), O(log n), O(n), O(n log n), and O(n²).

## Features
- Visualizes different time complexities.
- Interactive slider to change the value of n.
- Dynamic graph updates.
- Simple and user-friendly interface.

## Technologies Used
- Flask
- HTML
- CSS
- JavaScript
- Chart.js

## How to Run
1. Install Flask:
   pip install flask

2. Run the application:
   python app.py

3. Open your browser and visit:
   http://127.0.0.1:5000
